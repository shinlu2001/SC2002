# HDB BTO Management System

This Java-based project simulates the core processes of Singapore's HDB BTO application and management system. It supports roles like Applicants, HDB Officers, and Managers.



## Technologies
- **Java 17+**
- **Text-based UI (Console)**
- **CSV for persistence**
- **UML sequence diagrams**
- **Visual Paradigm**

---

## Project Structure
```
SC2002/
├── Project/
│   ├── boundary/         # UI layer
│   ├── control/          # Controllers
│   ├── entity/           # Domain models (Applicant, Officer, Project...)
│   ├── files/            # CSV data files
│   └── util/             # Utilities
├── enums/               # Enum types (Statuses, Roles, etc.)       
└── README.md
```
